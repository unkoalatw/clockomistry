# Clockomistry 更新日誌 - 介面舒適度與效能雙重優化

本文件詳述了針對 **Clockomistry** 依據網站體驗報告進行的最新一次核心改版。主要針對「電腦/手機雙平台的版面配置」與「觸控回饋」進行深度優化！

## 📱 1. 電腦與手機版面配置大翻新 (Responsive Layout)
先前的設計在「電腦端顯得螢幕兩側太空曠、手機端又顯得太擁擠」，為此我們進行了結構級的調整：

*   **釋放電腦螢幕空間**：
    *   將計時器 (Timer)、番茄鐘 (Pomodoro) 與碼錶 (Stopwatch) 等應用的最大寬度由 `max-w-lg` 大幅擴大為 `max-w-3xl`。
    *   這使得電腦使用者不再覺得視覺被侷限在中央窄窄的一條，現在所有數字、按鈕與進度環都能完美伸展並利用大螢幕的優勢。
*   **優化手機端兩側留白**：
    *   新增了預設左右內邊距 (`px-4`)，讓畫面邊緣不會因為緊貼螢幕而產生擁擠感，呼吸感大幅提升。

## 👆 2. 使用者體驗：放大點擊區域與觸控回饋 (UX Touch Targets)
為解決手機端「按鈕難按」的問題，帶來了實體 APP 級別的順暢體驗：

*   **指腹友善的超大按鈕**：
    *   「開始/暫停」與「重置」等核心操作按鈕，其尺寸從先前的原生大小強制放大至 **64x64px**（`w-16 h-16`），在平板與電腦上甚至來到 **80x80px**。完全符合且超越建議的 44x44px 原則。
    *   底部的「導覽列 Navigation」圓形工具按鈕也全部統一放大至 `w-12 h-12`，保證盲按也不會失誤。
*   **強烈的「按壓回饋 (Haptic-like Visuals)」**：
    *   所有重點按鈕同步加上了 `active:scale-90` 以及 `active:bg-white/30`。
    *   當手指按下去的瞬間，按鈕會有明顯的「縮小反彈與背景發亮」效果，創造出「清脆的點擊手感」。

## 🚀 3. 快取策略與專業效能 (Progressive Web App)
*   **離線與秒開體驗**：Service Worker 的 `sw.js` 已更新至 `v8` 版快取。我們將最新的優化字體（`.woff2`）、新舊版音效（`.mp3`, `.ogg`）全面預防性加入緩存中。這代表使用者第二次打開時，不會受到網路慢的影響，圖示與字體瞬間載入。
*   **JavaScript 極致壓縮**：套用了 Babel `--minified` 指令，大幅削減 `script.dist.js` 負擔。

## 🍎 4. 蘋果設備相容性 (Apple Audio Fallback)
*   已在前端代碼與快取名單中加入了雙軌並行的音訊策略。系統會先探測 `audio.canPlayType`，如果探測到 Safari 瀏覽器，就會自動替換播放 `.mp3` 檔案。（註：需確認您已手動將 MP3 備用音效放到 `/public/audio/` 資料夾下）。

## ✨ 最新追加功能清單 (New Features)
1.  **💾 本地設定備份 (Local Config Backup)**：現在改為匯出 JSON 設定檔，方便你手動保存、備份或在不同裝置間匯入自訂主題、紀念日與所有設定。
2.  **📍 智慧時區建議 (Smart Timezone Advice)**：在世界時鐘內以綠、黃、紅燈標示，一秒判斷該城市是否適合跨國會議。
3.  **✨ 粒子背景特效 (Particle Backgrounds)**：跟隨漸層主題自動調色的動態漂浮粒子系統。
4.  **⌨️ 全域快捷鍵控制 (Keyboard Shortcuts)**：按下空白鍵即可無縫控制番茄鐘、計時器、與碼表的播放與暫停。
5.  **📅 ics 格式導出 (Calendar Export)**：在紀念日功能中可一鍵下載 `.ics` 檔並加入 Google Calendar 行事曆。
6.  **⏳ 堆疊序列計時器 (Timer Sequences)**：全新的應用板塊，可自由設計多層次的連貫任務倒數，並具備自動接續播放功能。
7.  **🔋 自動智慧節能模式 (Eco Mode)**：當視窗隱藏或螢幕保護啟動時，自動暫停動畫與重繪以節省筆記型電腦電池。
8.  **🌓 自動深夜模式 (Schedule Dark Mode)**：偵測系統時間並於 18:00 - 06:00 之間自動調暗至 Modern 深色主題。
9.  **📦 微縮模式 (Mini Overlay)**：一鍵彈出精簡版小視窗，支援釘選置頂當作桌面小工具使用。
